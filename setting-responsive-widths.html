Setting Responsive Widths

By creating a container using a percent width that is less than 100, you can then add equal padding to either side to ensure that the content within that container will always have a decent amount of spacing. The amount of spacing will differ between devices when using percents. Note that this works best when applying the width and padding to parent container, rather than the children.

The Snip:

HTML

CSS

<div class="cont"> <div class="content"></div> </div>

.cont{ width: 90%; /* width must be less than 100% */ height: auto; /* height will be determined by contents */ padding: 5%; /* 5% padding on left and right + 90% width makes 100% */ /* you can set padding-top / padding-bottom afterwards if you want those adjusted */ }

Copy

Defects:

  • Works best when used with percents, avoid pixel values in most cases.
  • Does not always work with IE. Sometimes padding is ignored if child also has padding based on percents.
